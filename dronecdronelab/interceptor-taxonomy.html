Conceptual Definition (Draft)

Two distinct engagement paradigms are identified. Organic Detection-Engagement (ODE) refers to a framework in which the interceptor UAV autonomously detects, tracks, and engages the target drone using onboard sensors, forming a self-contained sensor-to-shooter loop. Externally Cued Engagement (ECE) refers to a framework in which target detection is performed by off-board sensor networks, with the interceptor assuming autonomous control only upon receiving an external cue and achieving proximity to the target.
